# Path Manipulation 

[app.py](./pythonapp/pathmanipulation/src/app.pysrc/app.py)


## Installation

1. Clone the repository:
```sh
git clone https://github.com/sahildari/secure-coding-examples
cd 'Path Manipulation'
```
2. Install the package:   
```sh
pip install .
```
## Usage
1. Run the Flask app
```sh
python run.py
```
2. Open in Browser:
```
http://127.0.0.1:5000
```
